Aprenda a Importar e Preparar Dados para Machine Learning

Este curso ensina como importar dados de diferentes fontes e realizar pré-processamento para modelos de Machine Learning.

O que você aprenderá:

  • Importação de dados de arquivos CSV, Excel e bancos de dados
  • Tratamento de valores ausentes e outliers
  • Normalização e padronização de dados
  • Codificação de variáveis categóricas
  • Automação de pipelines de dados
